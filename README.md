# NativPlayer Open Source Package

This package is the English open-source distribution of NativPlayer.

## Contents

- Application source code
- Build scripts
- Tests
- UI assets
- Documentation

## Not Included

- Prebuilt EXE files
- `libmpv-2.dll`
- Downloaded runtime binaries
- Build logs and temporary artifacts

## Notes

- End-user runtime acquisition happens at first launch through NativPlayer's managed runtime flow.
- If you build and redistribute with `libmpv-2.dll`, you must comply with the relevant `libmpv` / mpv license terms.

References:

- mpv upstream: <https://github.com/mpv-player/mpv>
- Windows build release: <https://github.com/zhongfly/mpv-winbuild/releases/latest>
